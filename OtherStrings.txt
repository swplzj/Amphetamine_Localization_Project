EXAMPLES:

@"Your Mac's battery is below %@."
THE %@ CHARACTER SET IS A PLACEHOLDER FOR THE BATTERY PERCENT

THE %li%% CHARACTER SET IS A PLACEHOLDER FOR THE BATTERY PERCENT
